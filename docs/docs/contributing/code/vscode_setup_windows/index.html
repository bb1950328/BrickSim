Using VS Code on Windows

  1. add C:\msys64\mingw64\bin to PATH
  2. Download and install VS Code from here if you haven't already
  3. Install the extensions ms-vscode.cpptools and ms-vscode.cmake-tools
  4. Open the BrickSim folder in VS Code. you should get the notifications like in the following screenshot (at least the first one):

    click "yes" on the first notification, we'll deal with the second one later.
  5. Now it's time to adjust some settings:
    1. Press Ctrl + Shift+ P and type "preferences: Open Settings (JSON)"

      Hit Enter or click the first result.
